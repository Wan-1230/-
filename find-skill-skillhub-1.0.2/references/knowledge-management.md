# knowledge-management · 知识管理（Knowledge Management）

面向知识沉淀与利用：知识库问答、资料整理、检索、摘要、笔记、归档、文档问答、长文阅读。

过滤用法：`?category=knowledge-management`。下列二级类目**仅用于理解边界与生成关键词**，不能作为 `?category=` 的值。

| 二级 key | 中文 | English |
|----------|------|---------|
| `knowledge-base-qa` | 知识库问答 | Knowledge Base QA |
| `knowledge-organize` | 资料整理 | Material Organizing |
| `knowledge-retrieval` | 信息检索 | Information Retrieval |
| `knowledge-summary` | 摘要总结 | Summarization |
| `knowledge-notes` | 笔记整理 | Note Taking |
| `knowledge-archive` | 知识归档 | Knowledge Archiving |
| `knowledge-doc-qa` | 文档问答 | Document QA |
| `knowledge-long-reading` | 长文阅读 | Long-form Reading |
